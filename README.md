# 표준 유지보수

이 저장소는 제정된 표준들의 제개정 이슈 도출을 위한 의견과 정보를 공유하기 위해 만들어 졌습니다. 질의 사항을 비롯한 의견, 아이디어 공유를 원하시면, [이슈](https://github.com/smartfarmKR/standards/issues)에 글 올려 주시기 바랍니다.
